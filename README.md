CPCS / CISRS / NPORS card verification, BS 7121 lift plans, mobile crane / tower crane operator certification.

## Install

```bash
pip install crane-hire-cpcs-mcp
```

## Tools

| Tool | Purpose |
|------|---------|
| `verify_cpcs_card` | CPCS card validity check by card number |
| `generate_lift_plan` | BS 7121 lift plan template generator |
| `appointed_person_check` | BS 7121-1 Appointed Person duties checklist |
| `crane_categories` | Mobile crane / tower crane / lorry loader category list |
| `loler_thorough_examination` | LOLER thorough examination reporting requirements |
